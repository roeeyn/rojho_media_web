You are designing and implementing a **premium, high-conversion landing page** for **RoJho Media**.

## 1) Brand context (must understand first)

**RoJho Media** is a creative + strategic media brand focused on helping founders, creators, and modern businesses communicate with clarity, authority, and emotion.  
It blends:
- Strategic messaging
- Visual storytelling
- Conversion-focused digital experiences

The brand is not “generic agency.” It should feel:
- Bold
- Elegant
- Modern
- Human
- Performance-oriented

**Core promise:** We transform ideas into media experiences that build trust, elevate brand perception, and drive measurable business results.

## 2) Audience

Primary audience:
- Startup founders
- Personal brands
- Creative entrepreneurs
- SMBs that need better brand positioning and online conversion

They usually struggle with:
- Inconsistent brand image
- Weak messaging
- Beautiful visuals but low conversion
- No clear “story to sale” funnel

## 3) Page objective

Main objective of this landing page:
1. Establish premium credibility fast
2. Explain RoJho Media’s value clearly
3. Drive user to a primary CTA (book call / start project / contact)

## 4) Visual direction

Design style:
- Premium dark-light contrast
- Cinematic gradients, subtle glow, clean geometry
- Strong typography hierarchy
- Spacious layout, modern cards/sections
- Micro-interactions that feel polished (not gimmicky)

Mood keywords:
- Strategic
- Confident
- Creative
- Elegant
- High-performance

Avoid:
- Template-looking layouts
- Overloaded sections
- Stock-agency clichés
- Excessive animations that hurt clarity

## 5) Suggested structure (conversion-first)

1. **Hero**
   - Strong headline (clear value proposition)
   - Subheadline (who + outcome)
   - Primary CTA + secondary CTA
   - Trust hint (results, clients, credibility)

2. **Problem / Pain**
   - 3 core pain points audience faces

3. **Solution**
   - How RoJho Media solves those pains
   - Clear framework (strategy → creative → execution)

4. **Services / Offer Stack**
   - Brand strategy
   - Content/media production
   - Landing pages / digital presence
   - Campaign creative systems

5. **Proof / Results**
   - Testimonial placeholders
   - KPI/result blocks (sample data placeholders allowed)

6. **Process**
   - 3–5 step process with clear outcomes

7. **Final CTA**
   - Strong close
   - Contact options

8. **Footer**
   - Brand, links, social, contact

## 6) Tone of copy

Voice:
- Direct, confident, strategic
- Warm and human (not robotic)
- No fluffy marketing jargon

## 7) Technical expectations

- Build a responsive React landing page (production-ready structure)
- Clean component structure
- Accessibility basics (semantic sections, aria labels where needed, contrast)
- Reusable UI components
- Keep performance in mind (optimized assets, minimal bloat)

## 8) Deliverable requested

Create:
1. A complete landing page UI with premium visual polish
2. Thoughtful placeholder copy aligned with brand context
3. Clean, maintainable code and sectioned components

Important: Every design decision should support trust + clarity + conversion.
